Outputs 1 if the **`IN0`** input value is less than or equal to the **`IN1`** input value, otherwise output 0.

## Socket information:
- **`IN0`**: Value A
- **`IN1`**: Value B
- **`OUT0`**: IF (A &lt;= B) THEN 1 ELSE 0

Inverts **`IN0`** and writes it to **`OUT0`**. If the input signal is raw, 1 - **`IN0`** is written to **`OUT0`**.

## Socket information:
- **`IN0`**: Input signal
- **`OUT0`**: Inverted **`IN0`**
